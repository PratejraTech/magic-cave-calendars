## Getting Started

### 1. Account Setup
1. Visit the website and click **"Get Started"**
2. Create an account with your email and password
3. You'll receive a confirmation email (check your spam folder!)
4. Sign in to access your dashboard

### 2. Child Profile Setup
1. Click **"Create New Calendar"**
2. Enter your child's name
3. Choose a persona for the chat feature:
   - **Mummy**: Warm, nurturing personality
   - **Daddy**: Fun, playful personality
   - **Custom**: Write your own persona description
4. Upload a hero photo of your child (optional but recommended)
5. Select a theme for the calendar (you can change this later)

## Troubleshooting

### Common Issues

**Calendar Won't Load:**
- Check that it's December 1-24
- Verify the share URL is correct
- Try refreshing the page

**Photos Not Uploading:**
- Check file size (max 5MB)
- Verify format (JPG, PNG, GIF)
- Try a different browser

**Chat Not Responding:**
- Check internet connection
- Try refreshing the chat window
- Contact support if issues persist

**Videos Not Playing:**
- Verify YouTube URL is correct
- Check if video is age-restricted
- Try a different video
